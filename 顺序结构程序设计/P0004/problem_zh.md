**题目描述**

　小飞侠无意中发现 7254 是一个不寻常的数，因为它可以表示为 7254 = 39 x 186，这个式子中 1~9 每个数字正好出现一次,输出所有这样的不同的式子（乘数交换被认为是相同的式子)

结果小的先输出,结果相同的，乘数较小的优先输出；

**输入格式**

null

**输出格式**

每个等式占一行

**输入样例：**

```
null
```

**输出样例：**

```
4396 = 28 x 157
5346 = 18 x 297
5346 = 27 x 198
5796 = 12 x 483
5796 = 42 x 138
6952 = 4 x 1738
7254 = 39 x 186
7632 = 48 x 159
7852 = 4 x 1963
```
